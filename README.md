# Алгоритм Фишера-Йетса: Реализация и Визуализация

Данный репозиторий содержит исходный код курсовой работы по теме «Алгоритм Фишера-Йетса», выполненной в НИТУ «МИСИС» (кафедра АСУ).

## О проекте
Проект представляет собой программную реализацию классического алгоритма случайной перестановки Фишера-Йетса (алгоритм Кнута) на языке Python. Основная цель — демонстрация эффективности алгоритма и наглядная визуализация процесса тасования элементов «на месте».

## Ключевые особенности
- **Эффективность**: Алгоритм работает с временной сложностью $O(N)$ и пространственной сложностью $O(1)$.
- **Консольное логирование**: Пошаговый вывод операций обмена элементов для контроля корректности работы.
- **Графическая визуализация**: Динамическое сравнение исходного и перемешиваемого массивов с использованием библиотеки `matplotlib`.
- **Использование генераторов**: Реализация через `yield` для покадровой отрисовки анимации.

## Структура проекта
- `main.py` — основная логика алгоритма и блок консольного тестирования.
- `visualize.py` — скрипт для запуска графической визуализации с двумя диаграммами (эталон и процесс).
